
КОМПИЛЯЦИЯ Eureka
================

Зависимости:

1. Компилятор C++ (GNU's G++) и сопутствующие инструменты
   пакеты: g++ binutils

2. GNU make
   пакеты: make

3. FLTK 1.3 
   вебсайт: http://www.fltk.org/
   пакеты: libfltk1.3-dev

   Вам также понадобятся: libxft-dev
                          libxinerama-dev
                          libfontconfig-dev
                          libjpeg-dev
                          libpng-dev

4. zlib
   вебсайт: http://www.zlib.net/
   пакеты: zlib1g-dev

5. Утилиты XDG
   (нужны только для Linux, для установки ярлыка на рабочий стол и добавление в меню)
   пакеты: xdg-utils

Заметка: в Linux Mint 18.3 нужно доустановить libghc-glut-dev (но не факт :-) )

После установки всех зависимостей, введите комманду для сборки
исполняемого файла Eureka.

   make


Некоторые системы требуют дополнительные CFLAGS и/или LDFLAGS 
для того, чтобы найти необходимые библиотеки (особенно FLTK). Например,
эти условия нужны для нормальной сборки для Gentoo Linux:

   CFLAGS  += -I/usr/include/fltk-1
   LDFLAGS += -L/usr/lib64/fltk-1/



INSTALLING Eureka
=================

Для начала, убедитесь в том, что исполняемый файл Eureka был успешно собран.

Затем, получив права root (через команды 'sudo' или 'su'), введите следующую
команду в терминал:

   > make install

Она установит исполняемый файл редактора, файлы настроек игр и source-портов,
а также загрузит ярлык программы для рабочего стола и пункт в меню..



Настройка Eureka
=================

Eureka требует IWAD от одной из поддерживаемых игр, для
примера "doom2.wad".  Eureka ищет IWAD'ы при запуске,
однако, если ей не удаётся их найти, запускается диалоговое окно
'Менеджер wad-файлов' и Вы сможете указать редактору расположение IWAD файла,
он его запомнит и загрузит при следующем запуске.

Где Eureka ищет IWAD'ы:

  1. В папке "iwads" в корневой папке Eureka.  В Linux
     это обычно "~/.eureka/iwads".  Она создаётся автоматически
     при первом запуске Eureka (но если Вы создатите её сами - 
     хуже не станет).

  2. (a) В Linux: /usr/share/games/doom
     (b) В Windows: C:\DOOM and C:\DOOM2

  3. Если Вы запустили Eureka из командной строки и имеете source-порт с 
     настроенными переменными DOOMWADDIR или DOOMWADPATH,
     редактор подхватит их автоматически.

  4. Также Eureka ищет IWAD'ы в той папке, в которой вы находитесь в терминале.

  5. Наконец, Вы можете укаазть путь к IWAD файлу через напрямую через параметр --iwad.
